def hypothesis_maker.make_hypothesis (   request)

it returns the response to the state machine.

Parameters
request, HypothesisRequest
Returns
: res, HypothesisResponse This function handles the service callback. Depending on the received hint and the updated knowledge it returns a complete or incomplete hypothesis. First it checks if the hint is not malformed and then adds it to the onthology, reading if there is any complete hypothesis in it.
